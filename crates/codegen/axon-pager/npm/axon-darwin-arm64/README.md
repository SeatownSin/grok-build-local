# @axon-official/axon-darwin-arm64

Platform-specific binary for [`@axon-official/axon`](https://www.npmjs.com/package/@axon-official/axon) on darwin-arm64.

Do not install this package directly. Install the main package instead:

```sh
npm install -g @axon-official/axon
```

The main package will automatically pull the correct binary for your platform via `optionalDependencies`.
