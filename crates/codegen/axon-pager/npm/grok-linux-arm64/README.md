# @axon-official/grok-linux-arm64

Platform-specific binary for [`@axon-official/grok`](https://www.npmjs.com/package/@axon-official/grok) on linux-arm64.

Do not install this package directly. Install the main package instead:

```sh
npm install -g @axon-official/grok
```

The main package will automatically pull the correct binary for your platform via `optionalDependencies`.
